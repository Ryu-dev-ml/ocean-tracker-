# OceanAI — Pollution Tracking System

Real-time ocean pollution tracking dashboard with AI detection, time travel, smart alerts and chatbot.

## Deploy on Vercel (5 minutes)

1. Upload this folder to GitHub
2. Go to vercel.com → New Project → Import from GitHub
3. Click Deploy
4. Share the link!

## Features
- Live ocean map with heat zones
- Time travel slider (past → present → future)
- Multi-class AI detection (oil spill, algae, plastic, chemical)
- Smart alert system
- AI chatbot
- Grad-CAM explainability panel
- Analytics dashboard
